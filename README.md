## Running

Edit the `.env` file to include paths. An example `.env` can look like:  
```bash
TYPEDEF_REPO=./ultimate-crosscode-typedefs
GAME_COMPILED_JS=./game.compiled.js
OUTPUT_GAME_COMPILED_JS=./game.compiled.typed.js
```

Then run:  

```bash
npm install
npm start
```
